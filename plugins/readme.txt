Updated Plugins for Phy2
By Mingze Dou, 2025

Original by Peter Petersen, Phy2 port by Thomas Hainmueller

NEW FEATURES:
1. ImprovedISIAnalysis (alt+i): Better spike conflict detection
2. StableMahalanobisDetection (alt+x): Interactive outlier removal
3. ReclusterUMAP (alt+k): Modern clustering with UMAP
4. GoodLabelsPlugin: Sorts clusters by quality

REQUIRES:
pip install pandas numpy scipy scikit-learn umap-learn
